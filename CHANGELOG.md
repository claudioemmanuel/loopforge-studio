# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-25

### Added

- Initial open-source release
- Kanban board for task management
- AI-powered brainstorming with chat interface
- Automated planning and execution
- Support for Anthropic, OpenAI, and Google AI providers
- GitHub integration for commits and branches
- Real-time progress updates via SSE
- Docker deployment with PostgreSQL and Redis
- Encrypted storage for API keys and tokens

### Changed

- Simplified to BYOK (Bring Your Own Key) model
- Streamlined onboarding flow

### Removed

- Managed billing (Stripe integration)
- Subscription-based features
